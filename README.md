# My_Expense-Analysis-Project
**Description:** Personal Monthly Expense Analysis with Python – Tracks spending patterns, highlights top expenses, and provides actionable budgeting insights.
## Dataset
- Source: Self-recorded expenses over 30 days
- Columns: Date, Category, Description, Amount, Payment_Mode
- Categories: Food, Rent, Utilities, Shopping, Travel, Education, Others
## Analysis Highlights
- Total Expenses: ₹9029
- Highest Spending Category: Food
- Fixed Costs: Rent, Utilities
- Non-Essential Spending: Shopping, Others
- Payment Mode: UPI transactions more frequent than Cash
## Insights
- Food dominates monthly spending.
- Fixed costs (Rent & Utilities) are significant.
- Non-essential expenses can be optimized to save money.
- Digital payments (UPI) preferred over Cash.
- Consistent daily spending helps in predictable budgeting.
## Recommendations
- Set monthly limits per category.
- Track expenses weekly.
- Reduce non-essential purchases.
- Allocate saved money to investments or learning.
## Future Enhancements
- Predict next month’s expenses using ML.
- Build an interactive finance dashboard.
- Add anomaly detection for unusual spending.
- Automate data input via mobile app.
